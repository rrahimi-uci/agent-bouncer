# agent-bouncer ‚Äî paper

ACM-formatted (`acmart`, sigconf) measurement study of a laptop-trained SmolLM3-3B safety guard.

## Files
- `safety_guard_operating_point_artifacts.tex` ‚Äî the paper (ACM `acmart`, `sigconf` + `nonacm`).
- `safety_guard_operating_point_artifacts.pdf` ‚Äî compiled output.
- `refs.bib` ‚Äî 32 fact-checked references (verified arXiv IDs).
- `figures/` ‚Äî vector PDF figures (generated from result JSON; Okabe-Ito colorblind-safe palette):
  - `fig1_inhouse_auprc.pdf` ‚Äî in-house pooled AUPRC (guard vs open guards).
  - `fig2_novel_auprc.pdf` ‚Äî novel held-out AUPRC (base > tuned > Llama-Guard).
  - `fig3_operating_point_flip.pdf` ‚Äî native-F1 vs AUPRC vs matched-FPR ranking flip.
  - `fig4_base_vs_tuned.pdf` ‚Äî per-benchmark base‚Üítuned F1 delta.
  - `fig5_pareto.pdf` ‚Äî F1 vs single-request latency Pareto.
- `DRAFT.md` ‚Äî the prose source the LaTeX was converted from (kept for editing).
- `metrics_survey.md` ‚Äî survey of related-work evaluation metrics (background for the eval protocol).
- `tables/operating_point_fairness.md` ‚Äî the operating-point-fairness table.

## Build
```bash
make            # compile safety_guard_operating_point_artifacts.tex -> safety_guard_operating_point_artifacts.pdf (uses tectonic; auto-fetches acmart)
make figures    # regenerate figures/ from the result JSON
make clean
```
Or directly: `tectonic safety_guard_operating_point_artifacts.tex` (needs network on first run to fetch acmart).

## Provenance
All numbers trace to `notebooks/outputs/nb-smollm3-guard/*.json`. Producing scripts:
- `scripts/eval_corrected.py` ‚Üí `summary_corrected.json`, `preds_corrected.json`
  (in-dist-only calibration T=2.10/œÑ=0.59, matched-FPR@0.10, AUPRC, batch=1 latency).
- `scripts/eval_novel_gaps.py` + `scripts/verify_novel.py` ‚Üí `_cache_{guard,base,llama}_exp.json`,
  `summary_novel_full.json` (novel-set AUPRC + Optimal-F1; `verify_novel.py` reconstructs the
  gold/order deterministically and re-grounds base 0.886 / tuned 0.781 / llama 0.701).
- `scripts/score_base_inhouse.py` ‚Üí `base_smollm3_inhouse.json` (base continuous in-house scores).
- `scripts/recompute_base_vs_tuned.py` ‚Üí `base_vs_tuned_clean.json` (base-vs-tuned at CLEAN
  per-model in-dist calibration on the identical 2,018 rows: base 0.713 / tuned 0.794,
  Œî+0.081 [0.062,0.100]; base in-house AUPRC 0.696).
- `scripts/guard_eval_pipeline.py`, `scripts/eval_expanded_heldout.py` ‚Äî parameterized / earlier eval.

Figures are regenerated by `scripts/make_figures.py`.
